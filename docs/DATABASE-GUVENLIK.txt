═══════════════════════════════════════════════════════════════
  NEDEN GÜVENLİ?
═══════════════════════════════════════════════════════════════

1. ✅ AYRI KONUM
   • Database: AppData klasöründe (kullanıcı verileri)
   • Uygulama: Program Files klasöründe (uygulama dosyaları)
   • Bunlar TAMAMEN AYRI konumlar!

2. ✅ MSI INSTALLER SADECE UYGULAMA DOSYALARINI GÜNCELLER
   • Installer sadece Program Files'taki dosyaları değiştirir
   • AppData klasörüne DOKUNMAZ
   • Database dosyası aynen kalır

3. ✅ WINDOWS STANDARTI
   • Tüm Windows uygulamaları böyle çalışır
   • Kullanıcı verileri AppData'da korunur
   • Bu Microsoft'un önerdiği yöntem

═══════════════════════════════════════════════════════════════
  GÜNCELLEME SIRASINDA NE OLUR?
═══════════════════════════════════════════════════════════════

1. Eski uygulama kapatılır
2. Yeni .msi dosyası çalıştırılır
3. Windows Program Files'taki eski dosyaları siler
4. Yeni uygulama dosyalarını yükler
5. AppData klasörüne HİÇBİR ŞEY YAPILMAZ
6. Database dosyası aynen kalır
7. Yeni uygulama açılır → Database'i bulur → Tüm veriler yerinde!

═══════════════════════════════════════════════════════════════
  EKSTRA GÜVENLİK İÇİN (İSTEĞE BAĞLI)
═══════════════════════════════════════════════════════════════

Eğer %100 emin olmak istersen, güncelleme öncesi yedek al:

1. Database konumunu bul:
   • Ayarlar ekranından görebilirsin
   • Veya: %APPDATA%\com.bayer.pos\pos.db

2. Yedek al:
   • pos.db dosyasını kopyala
   • Güvenli bir yere kaydet (USB, Cloud, vb.)

3. Güncellemeyi yap

4. Kontrol et:
   • Uygulamayı aç
   • Ürünlerin hala orada olduğunu gör
   • Yedek dosyayı silebilirsin

  • Satır 32: Database.load('sqlite:pos.db')
  • Tauri plugin otomatik olarak AppData'daki dosyayı bulur
